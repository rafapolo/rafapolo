WebDocGraffiti

Plataforma Cartográfica de WebDocumentários

https://github.com/rafapolo/webdocgraffiti

Tecnologias, Frameworks e APIs

  • RubyOnRails
  • HAML para abstração de HTML (estrutura)
  • SASS para abstração de CSS (estilo)
  • Coffescript para abstração de JavaScript (controle)
  • Mozilla Popcorn.js para controlar timeline de videos
  • Twitter Bootstrap para Backend (admin)
  • SQLite3 como banco de dados (1000 queries/seg)

  • Google Maps API
  • Facebook API
  • Youtube API
  • Google StreetView API
